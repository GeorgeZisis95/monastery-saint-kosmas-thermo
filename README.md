# Monastery Website

A responsive React/Vite website built for a monastery, providing visitors with information, daily saints, events, and resources.

## Overview

This project delivers a clean, fast, and mobile-friendly interface with attention to accessibility and maintainability. It includes automatic fetching of the Orthodox saint of the day, content sections for history and spiritual life, and structured components for easy expansion.

## Features

- Responsive layout using CSS and modern utility patterns
- Hero section with call-to-action buttons
- Automatic retrieval of the Orthodox saint of the day
- About section formatted for readability on large and small screens
- Book section with expandable descriptions
- Clean component-based structure with React
- Vite development environment for fast builds and hot reload

## Tech Stack

- **React**
- **Vite**
- **JavaScript**
- **HTML/CSS**
